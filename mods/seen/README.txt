seen mod for minetest

This mod stores data of last occurence of every player online.
Data can be viewed by command /seen <name>.
Default automatic check is done every 60 seconds, this can be changed by:
seen.save_interval = <time in seconds> in minetest.conf

This mod needs 0.4.8 or some latest git versions of 0.4.7

Commands:

/seen <name>
