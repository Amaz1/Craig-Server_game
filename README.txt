Game used for Craig's Survival Server [Craig-Survival_game]
==========================================================

This is the game used by Craig's Survival Server

It is an edited version of minetest_game (https://github.com/minetest/minetest_game) with lots of other mods added to it.

For information about the server go here: https://forum.minetest.net/viewtopic.php?id=7010
